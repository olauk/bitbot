## Use as Extension

This repository can be added as an **extension** in MakeCode.

* åpne [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klikk på **Nytt prosjekt**
* klikk på **Utvidelser** i menyen under tannhjulet
* søk etter **https://github.com/olauk/bitbot** og importér

## Rediger dette prosjektet ![Bygg statusmerke](https://github.com/olauk/bitbot/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* åpne [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klikk på **Importer** og så på **Importér URL**
* lim inn **https://github.com/olauk/bitbot** og klikk på importér
